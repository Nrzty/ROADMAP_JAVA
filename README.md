# 🚀 Spring Boot Roadmap — Desenvolvedor Júnior

Uma ferramenta interativa para estruturar seu aprendizado em **Spring Boot** de forma ordenada e alinhada ao que o mercado realmente exige de um desenvolvedor júnior.

## 📋 Visão Geral

Este é um **roadmap interativo** que guia você através de 7 fases essenciais para dominar Spring Boot:

1. **Fundamentos Java** (3–4 semanas)
2. **Core do Spring & Spring Boot** (3–5 semanas)
3. **Spring MVC — APIs REST** (3–4 semanas)
4. **Persistência de Dados** (3–4 semanas)
5. **Segurança** (2–3 semanas)
6. **Testes** (2–3 semanas)
7. **Boas Práticas & Mercado** (Contínuo)

## ✨ Características

- ✅ **Progressão visual** com barra de progresso em tempo real
- 🔄 **Sistema de ciclo** para marcar tópicos: TODO → EM ANDAMENTO → FEITO
- 💾 **Persistência local** via LocalStorage — seu progresso é salvo automaticamente
- 🎯 **Priorização inteligente** com estrelas (1-5) indicando importância
- 🏷️ **Tags contextuais**:
  - **Obrigatório**: Essencial para todo Júnior
  - **Mercado**: O que recrutadores cobram em entrevistas
  - **Diferencial**: O que você dominar se destacará
- 📱 **Design responsivo** com tema dark moderno
- 🔗 **Fácil navegação** com fases expansíveis

## 🎨 Design

- **Palette**: Verde matrix (#00e676) sobre fundo escuro (#0a0f0a)
- **Tipografia**: Syne (títulos) + Space Mono (código/dados)
- **Grid background**: Visual cyberpunk com padrão grid sutil
- **Acessibilidade**: Contraste adequado e feedback visual claro

## 🚀 Como Usar

1. **Abra o arquivo** `index.html` em seu navegador
2. **Clique em cada tópico** para ciclar entre os estados:
   - ⬜ **A fazer** (vazio)
   - 🟨 **Em andamento** (~)
   - 🟩 **Concluído** (✓)
3. **Seu progresso é salvo automaticamente** no LocalStorage do navegador
4. **Use o botão Resetar** para zerar todo o progresso quando necessário

## 💡 Dica do Professor

> **Não tente absorver tudo de uma vez.** A lógica ideal é:
> 1. Domine **Java Core** antes de tocar no Spring
> 2. Um dev Júnior que entende bem OOP, coleções e exceções vale mais do que alguém que só sabe copiar anotações
> 3. **Foque na Fase 1 por pelo menos 3–4 semanas** antes de avançar

## 🎯 Roteiro de Aprendizado Recomendado

### Fase 1: Fundamentos Java (3–4 semanas)
Mande ver em OOP, coleções, exceções e lambdas. Isso é **inegociável**.

### Fase 2: Core do Spring (3–5 semanas)
Entenda IoC, injeção de dependência e auto-configuration. O coração do framework.

### Fase 3: APIs REST (3–4 semanas)
Comece a criar endpoints. Aqui você vira "desenvolvedor web" de verdade.

### Fase 4: Persistência (3–4 semanas)
Integre banco de dados. Spring Data JPA é seu melhor amigo.

### Fase 5: Segurança (2–3 semanas)
JWT e autenticação básica. Essencial em qualquer projeto real.

### Fase 6: Testes (2–3 semanas)
JUnit + Mockito. Júnior que testa se destaca no mercado.

### Fase 7: Boas Práticas (Contínuo)
Arquitetura em camadas, Git, Docker, documentação. O que separa Júnior de Pleno.
